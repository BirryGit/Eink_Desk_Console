## Data
Replace the names and the dates with your data, and you should be set. You may need to expand the functions if you have more values.

## App Scripts
This is the harder part. All you need to do is go to Extensions at the top and select APP Scripts. From there, paste the code, and it should work. Put a Script Property in settings with the name "API_TOKEN". Then from deploy, deploy it as a web app, making sure to allow all access. Then copy the link and put it into the code!

## Getting APIS
For most of these, they should be very easy to find the correct way to key keys, so there is no reason for me to get into that. 

For Home Assistant, it is more complicated. What you need to do is go to account settings, security, and then create a long-lived access token. You also need the base URL you use to access HA. Copy both of these things into the code, and you should be good.
